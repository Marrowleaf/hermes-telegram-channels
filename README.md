# 🔧 hermes-telegram-channels

A [Hermes Agent](https://hermes-agent.nousresearch.com/) skill for **telegram-channels** (social-media category).

## Installation

```bash
cd ~/.hermes/skills/social-media
git clone https://github.com/Marrowleaf/hermes-telegram-channels.git telegram-channels
```

## Usage

Once installed, this skill is automatically detected by Hermes Agent.

```
skill_view(name="telegram-channels")
```

## Structure

- `SKILL.md` — Full documentation
- `references/` — Reference materials
- `scripts/` — Utility scripts
- `templates/` — Templates
